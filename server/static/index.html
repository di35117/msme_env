Problem

In Indian banking, borrowers rarely state reality directly. MSME owners understate stress in Hindi/Hinglish ("sab theek hai sir, thoda time aur de dijiye"). Startup founders overstate health in English ("Series B almost closed"). The agent must decode hidden financial state from messages + behavioral proxies, then choose one of 21 intervention actions across a 36-month horizon — without an LLM judge.

Anti-Reward-Hacking Guarantees

Cluster-Cascade

−0.25 / cascade
Aggressive coercion on a high-centrality MSME triggers cascade defaults across linked accounts. This rules out the "be maximally aggressive, recover the loan, ignore the cluster" strategy.

SARFAESI on Startup

−0.15 / occurrence
SARFAESI is a collateral-recovery tool for asset-backed MSMEs. Using it on a startup is wrong-tool; extra penalty applied so the right move (request_investor_update_meeting) dominates in expectation.

Shortcut Penalty

capped at −0.25 / ep
Episode-end checks for no-op farming, malformed-JSON spam, action spamming, account thrashing. Reward gets clipped on degenerate behaviour even if every step's local reward is positive.
